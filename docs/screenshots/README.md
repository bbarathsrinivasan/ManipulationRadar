## Screenshot Guidelines

- **Resolution**: 1280x720 or higher
- **Format**: PNG for static images, GIF for animations
- **Theme**: Dark theme to match extension design
- **Content**: Show actual usage on ChatGPT or Claude

## Taking Screenshots

1. Open ChatGPT or Claude in Chrome
2. Load the Manipulation Radar extension
3. Use Chrome DevTools to capture screenshots:
   - Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
   - Type "Capture screenshot"
   - Select "Capture area" or "Capture full size screenshot"
